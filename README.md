## Installation

The preferred way to install this template is through [npm](https://www.npmjs.com/).

To install dependencies simply:

    $ npm install

## Quick Start

Serve with hot reload at localhost:8080.

    $ npm run dev

Build for production with minification.

    $ npm run build
